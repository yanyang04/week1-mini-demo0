# Week1 Mini Demo0 链上存证智能合约实训
Week1 Mini Demo0 链上存证智能合约实训作品，基于Monad Testnet测试网
## 1. 本次完成内容
基于Monad开发一套轻量级文件哈希链上存证Demo，完整覆盖合约开发、本地调试、测试网部署、钱包交互全流程：
1. 编写Solidity 0.8.x存证智能合约，实现文件哈希存储、哈希查询、时间戳记录核心逻辑
2. 编写Hardhat测试脚本，本地模拟链完成功能单元测试
3. 在Monad测试网完成合约部署，并通过钱包执行链上读写交互
4. 整理操作截图、链上交易凭证、完整说明文档上传GitHub归档

## 2. 真实链上操作部分（可在区块浏览器核验）
1. 合约部署：将编译后的合约字节码部署至Monad测试网，生成公开合约地址与部署交易哈希
2. 链上写入：调用合约storeFileHash函数，将测试文件哈希永久写入链上存储层，交易上链不可篡改
3. 链上读取：调用getFileInfo函数从链状态读取已存储哈希、区块时间戳，验证数据一致性
4. 所有操作均产生可查询的链上TxHash，附截图存放在仓库screenshot文件夹

## 3. AI辅助完成部分
1. Solidity合约基础存储结构、读写函数模板生成
2. Hardhat部署脚本、单元测试用例基础代码生成

## 4. 人工判断与手动修改（核心自主工作）
1. 安全逻辑补充：AI原始代码无重复哈希拦截逻辑，手动新增判断，避免重复上链浪费Gas
2. 语法版本适配：修正AI生成的旧版Solidity语法，统一适配0.8稳定版本，内置溢出防护
3. 测试网环境适配：替换默认RPC为Monad测试网节点，隔离私钥至.env文件并配置.gitignore过滤
4. 边界用例补全：手动新增空哈希、超长哈希异常输入测试，修复AI未覆盖的漏洞分支
5. 文档精简优化：删减AI冗余话术，按作业5项要求结构化排版，新增截图标注方便阅读
